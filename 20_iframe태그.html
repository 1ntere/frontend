iframe 사용하기

    iframe : 웹페이지나 웹에 있는 정보를 포함해서 보여주는 태그
    사용을 하는데 불편한 부분이 많기 때문에
    html5 부터는 사용을 많이 하진 않음
    html 을 포함해서 보여주기도 함

    src="           : 불러올 위치 설정
    width=""        : 가로길이 지정
    height=""       : 세로길이 지정
    frameborder=""  : 테두리 지정, 0으로 설정하면 테두리 없음
        
    iframe의 경우 외부에서 태그해서 볼 수 없도록 지정해 놓으면
    엑스박스로 되면서 홈페이지에서 연결을 거부했습니다. 가 표시됨
    외부에서 볼 수 있도록 지정해 놓으면
    html에서 src로 불러온 홈페이지를 볼 수 있음
