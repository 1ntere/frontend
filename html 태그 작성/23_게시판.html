A 태그로 이동하는 이미지 😊

    페이지 내부에서 점프하기
    페이지 내부에서 소비자나 개발자가 원하는 위치로 이동할 수 있음

    a태그 이외에 다른 태그들에는 id 속성을 이용해서 변수명을 지정

    a태그에서 href안에 #변수명을 작성해주면 내가 원하는 태그 위치로 이동할 수 있음

    사용예제
    img id=변수명 src=123.jpg
    a href=#변수명 id=변수명
