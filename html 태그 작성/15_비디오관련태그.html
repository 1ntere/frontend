비디오 태그

    video를 재생하는데 사용되는 태그
        controls : 재생, 일시정지, 음량조절 등의 기능을 활용할 수 있음
        autoplay : 자동재생
            muted : 비디오 음소거
            loop : 비디오가 끝나면 자동으로 반복해서 무한 재생

    source : 비디오 파일을 제공하기 위한 공간으로 실질적으로 비디오를 불러오는 공간
        src : 불러올 비디오의 현재 위치 설정
        type : 비디오 형식 지정
              필수로 작성해야 하는 것은 아니지만 오류 발생을 줄이기 위해서 작성

    사용예제
        video 태그 옆에 controls를 작성
        video 태그 안에 source를 작성
                
        video 태그 옆에 controls 또는 autoplay
            source 태그 옆에 src를 사용해서 불러올 비디오가 현재 있는 위치 설정
                            type를 사용해서 비디오 형식 지정
        
    iframe : 웹에서 또 다른 html을 태그하거나 비디오를 가지고 오는 태그
              저작권 문제로 외부에서 재생을 막았을 경우에는 실행할 수 없음
