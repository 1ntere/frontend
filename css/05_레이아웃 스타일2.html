태그의 여백 관련 스타일

    1. padding
        사람이 패딩을 입었을 때 몸이 커지는 것 처럼
        태그의 영역이 상하좌우 넓어짐
        내용과 선 사이의 공간
    2. border 테두리
        영역이 어디까지 인지 구분하기 위한 선
    3. margin 여백 간격
        다른 태그와 간격을 나타내는 영역

padding margin border 방향성

    000 -> padding, margin, border를 나타낸다고 가정할때

    000-top:크기;        -> 위쪽   방향 크기 지정
    000-bottom:크기;     -> 아래쪽 방향 크기 지정
    000-left:크기;       -> 왼쪽   방향 크기 지정
    000-right:크기;      -> 오른쪽 방향 크기 지정

    단 border(선) 속성 값은 [크기 모양 색상]형식
    예를 들어
    테두리 중 오른쪽 선만 도트모양으로, 10px, 빨간색으로 하고 싶다면
    border-right: 10px dotted red;

padding, margin, border-width 크기 한번에 지정하기

padding | margin | border-width : 속성 값 크기 설정

속성 값 1개 : 상하좌우
padding: 20px;
    /*상하좌우로 20px 패딩을 주라는 의미*/
margin: 20px;
    /*상하좌우로 20px 다른 태그와의 간격을 주라는 의미*/

속성 값 2개 : 상하 좌우
padding: 20px 10px;
    /*상하 20px 좌우 10px 패딩을 주라는 의미*/
margin: 20px 10px;
    /*상하 20px 좌우 10px 다른 태그와의 간격을 주라는 의미*/

속성 값 3개 : 상 좌우 하
padding: 20px 10px 30px;
    /*상 20px 좌우 10px 하 30px 패딩을 주라는 의미*/
margin: 20px 10px 30px;
    /*상 20px 좌우 10px 하 30px 다른 태그와의 간격을 주라는 의미*/

속성 값 4개 : 상 우 하 좌(시계방향)
padding: 20px 10px 30px 40px;
    /*상 20px 우 10px 하 30px 좌 40px 패딩을 주라는 의미*/
margin: 20px 10px 30px 40px;
    /*상 20px 우 10px 하 30px 좌 40px 다른 태그와의 간격을 주라는 의미*/

margin: 0 auto;

다른 태그와의 간격을 자동으로 지정해서 가운데 정렬하게 만듦

기본적으로 자동정렬은(=가운데 정렬) display: block; 으로 자주 사용

inline, inline-block은 글자취급이기 때문에 text-align: center;이용
